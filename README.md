# cell70horn.github.io

Compute a file's MD5, SHA-1, SHA-256 and SHA-512 checksum locally in your browser. Files are never uploaded; everything runs client-side. Includes a transfer-time and data-unit converter to verify downloads from Quark NetDisk and other clouds.

Live: https://cell70horn.github.io/

Client-side file hash (MD5/SHA-1/SHA-256/SHA-512) + transfer-time / unit converter. 12 languages. MIT License.
